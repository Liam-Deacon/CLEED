77  information about the composite layer. Used in this function:

81  of all atoms.
82  real a_lat[5]: lattice vectors:
83  a_lat[1] = a1_x, a_lat[2] = a2_x,

94 
95  R/T(k'k) = pref_i * |k'><l,m|M(lm,l'm')|l'm'><k| + dkk'
96 
97  Ylm = |k'><l,m| = Ylm(k') (k_max rows) x ((l_max+1)^2 columns)
98 
99  pref_i = -i*16*PI^2 * EXP/ (A*k'z+)
100  EXP = exp[i(-k(+)r1 + k'(+)rN)] for ++
101  exp[i(-k(+)r1 - k'(-)r1)] for -+
102  exp[i(+k(-)rN - k'(-)r1)] for --
103  exp[i(+k(-)rN + k'(+)rN)] for +-
104 
105  r1 = (0, 0, (atoms+0)->pos[3]).
106  Projection of the origin to the atomic subplane with the greatest
107  z coordinate (atom No. 0):
108 
109  rN = (0, 0, (atoms+n_atoms-1)->pos[3]).
110  Projection of the origin to the atomic subplane with the largest
111  z coordinate (atom No. n_atoms-1):
112 
113  FUNCTION CALLS

571 /**********************************************************************
572  Prepare matrices for conversion into plane waves:
573
