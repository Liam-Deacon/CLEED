32 
33  Calculate scattering matrix for Bravais layer.
34 
35  INPUT:
36 
37  mat * p_Tpp, p_Rpm - (output) pointers to composite layer diffraction

40  Rpm k(-) -> k(+) and k(+) -> k(-) (reflection matrix)
41 
42  leed_var v_par - (input) parameters which vary during the energy
43  loop. Used: k_in (parallel components of the incoming beam),
44  l_max (max. l quantum number),
45  tl (atomic scattering factors),

47  be considered).

51 
52  leed_atom * atoms: scattering properties of the atoms.
53  real a_lat[5]: lattice vectors:
54  a_lat[1] = a1_x, a_lat[2] = a2_x,
55  a_lat[3] = a1_y, a_lat[4] = a2_y;
56  real rel_area: area of the unit cell relative to 1x1.
57 
58  leed_beam_t * beams - (input) additional information about the k
59  vectors involved.
60  Used: k_r, k_i: wave vectors,
61  set: current beam set,
62 
63 
64  DESIGN
65 
66  R/T(k'k) = pref_i * |k'><l,m|M(lm,l'm')|l'm'><k| + dkk'
67 
68  Ylm = |k'><l,m| = Ylm(k') (k_max rows) x ((l_max+1)^2 columns)
69 
70  FUNCTION CALLS
71 
72  leed_ms_lsum_ii

75  RETURN VALUES:
76

117  - if not, recalculate lattice sum, scattering matrix, and spherical

196  Gii = mattrans(Gii, Gii);
197

202  }
203  }

206  *p_Tpp = matmul(*p_Tpp, Gii, Yin_p);
