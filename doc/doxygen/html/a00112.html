Detailed Description

Author
Georg Held g.hel.nosp@m.d@re.nosp@m.ading.nosp@m..ac..nosp@m.uk

Calculates the mean square displacement using leed_inp_debye_temp(). This function is used in both cleed and csearch programs.

Note
EXIT_ON_ERROR should be defined while compiling if you wish the leed_inp_debye_temp() function to stop program execution on failure.
Additional output is available by adding the defines WARNING , CONTROL and ERROR during compilation.
