39 /************************************************************************
40 
41  Calculate an array of temperature dependent atomic scattering factors.
42 
43  INPUT:
44 
45  mat tl_t - (input) The function writes the output temperature dependent
46  scattering factors into its first argument. If tl is NULL,
47  the structure will be created.
48  mat tl_0 - (input) array containing scattering factors for T = 0. tl_0
49  will be stored before tl_t is calculated, therefore tl_0 can
50  be equal to tl_t.
51 
52  real dr2 - (input) mean square displacement (not isotropic vibrational
53  amplitude) at the given temperature:
54  <(dr)^2>(T) = ux^2 + uy^2 + uz^2 = 3ux^2
55  (since ux^2 = uy^2 = uz^2).
56  real energy - (input) energy (real part).
57 
58  int l_max_0 - (input) max l quantum number of input scattering factors.
59  int l_max_t - (input) max required l quantum number of output scattering
60  factors.
61 
62  RETURN VALUES:
63 
64  mat tl_t - set of temperature dependent scattering factors
65  NULL if any error occured (and EXIT_ON_ERROR is not defined)
66 
67  DESIGN:
68 
69  According to formula (23/p. 29) in VHT:
70 
71  t(l1,T) = sum(l2,l3) { i^l3 * exp(-4aE) * J(l3)(-4iaE) * t(l2,0) *
72  sqrt[ 4pi (2*l2+1) * (2*l3+1) / (2*ll+1) ] *
73  cg(l3,0; l2,0; l1,0) }
74 
75  a = 1/6 <(dr)^2>(T).
76  E = E_vac - V0 (kinetic energy inside the crystal).
77  J(l2) = bessel function.
78 
79 *************************************************************************/
