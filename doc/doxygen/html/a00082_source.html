29  Purpose:
30  getint serves to :
31  - integrate over an elliptical spot area ( axes defined as scale->xx/yy )
32  - integrate over an elliptical background area (axes = VERH*scale->xx/yy )
33  - if ( bg == 1 ) -> subtract background from spot intensity
34  - calculate S/N ratio :
35  - set spot[i].control bit SPOT_GOOD_S2N if spot has good s/n ratio
36  - set spot[i].control bit SPOT_OUT if spot touches boundaries :
37  acci = percentage of spot area that must be within visible LEED-screen
38  accb = percentage of background area - " -

76 /****************************************************************************
77 
78  Search for the points which lie inside the (rotated) ellipses around the
79  spot maxima and sum up the intensities for each spot ( list spot ).
80 
81  A point (x,y) being inside an ellipsis with half axes h and v is defined by
82  the following equation:
83  x*x/(h*h) + y*y/(v*v) <= 1. (1)
84  If the coordinate system is rotated about (-angle), x and y are transformed:
85  x(new) = x*cos(angle) + y*sin(angle) (2)
86  y(new) = -x*sin(angle) + y*cos(angle)
87  The ellipsis equation (1) then transforms as:
88  x*x* x_fac + y*y* y_fac + x*y * xy_fac <= 1. (3)
89 
90  spot.s2u is temporarily used as background.
91  spot.s2n is temporarily used as SQUARE(background)

148  if ( b_norm < TOLERANCE || b_norm > 1./TOLERANCE ) continue;
149 
150  sgma =sqrt( spot[k].s2n -SQUARE(spot[k].s2u)/b_count[k] )/b_count[k];
151  sgmb =spot[k].intensity/e_count[k] -spot[k].s2u/b_count[k];

154 
155  if ( bg == BG_YES ) {

157  spot[k].intensity -= spot[k].s2u;
158  spot[k].s2u = spot[k].intensity / spot[k].s2u;
159  spot[k].intensity /= use_cur;

161  else {

165  }
