Detailed Description

Contains all the function prototypes for the search program.

Author
Georg Held g.hel.nosp@m.d@re.nosp@m.ading.nosp@m..ac..nosp@m.uk
Liam Deacon liam..nosp@m.deac.nosp@m.on@di.nosp@m.amon.nosp@m.d.ac..nosp@m.uk

There are two code interfaces for the search routines. The original interface uses calls to routines taken from the copyrighted "Numerical Recipes in C", therefore for the open source version of the code there is a new interface to the GNU Scientific Library - this is a major re-write of parts of the SEARCH code and should therefore be considered as highly experimental.

Warning
The open source version of the search program, enabled when compiling with _USE_GSL defined, is highly experimental and needs significant testing.

int sr_rdinp ( const char *  inp_file)

Read input for search from input file (and *.bul)

Parameters
[in]inp_fileinput file containing start geometry and control parameters for SEARCH.

Unlike in the CLEED program this input routine does not convert the parameters into atomic units (BOHR and HARTREE).

When reading the input file the currently recognized identifiers are: 'a1:','a2:' bulk unit cell parameters. 'b1:','b2:' overlayer unit cell parameters. 'm1:','m2:' superstr. matrix. 'it:','ip:' angles of incidence.

Warning
The above parameters a1, a2, b1, b2, m1, m2, and ip, it will be searched for in inp_file as well as in <sr_project>.bul. If they are found in both files inp_file is given priority.

'po:' position and type of bulk atoms.

'rm:' minimum radius. ( name r_m : type name, min. radius) 'zr:' range of z values. ( z_min z_max : min./max. z value)

'sr:' rotation symmetry ( n x_a y_a : degree, x/y coord of axis). 'sm:' mirror symmetry ( x_d y_d x_p y_p : direction, x/y coord of point in mp.). 'sz:' 1 = optimize only z coordinates; 0 = optimize all coordinates. 'sa:' 1 = angle search is on; 0 = angle search is off.

'spn:' number of parameters (n_par) 'spp:' parameter reference list (n_atom x/y/z par_ref_list) 

    An atomic coordinate x is calculated as x_0 plus the sum over
    x_par[i] * par[i], where x_par is the parameter reference list
    for this particular atom and par the list of (all) parameters.

'rft:' R factor type (input for R factor program) 'rfr:' shift range (input for R factor program)

Each entry of atoms will contain:

  • type: reference to list types.
  • x,y,z: atom positions
  • dr: vibrational amplitude ( )
Note
sr_atoms and sr_search will be allocated in this function.
Returns
number of parameters.
Return values
ref SR_FAILURE if failed and EXIT_ON_ERROR is not defined.
Note
the exit code search_error will be be given if the function fails and EXIT_ON_ERROR is defined.
