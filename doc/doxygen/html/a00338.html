Detailed Description

Search according to simulated annealing (simplex) method.

Perform a search according to the simulated annealing (simplex) method. Driver for routine sr_amebsa_gsl() (using GNU Scientific Library).

Warning
This routine needs to be tested thoroughly and as such is under development and should be considered as highly experimental (unstable).
