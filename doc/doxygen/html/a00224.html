Detailed Description

Master include for patt - LEED pattern visualisation program.

patt simulates Low Energy Electron Diffraction (LEED) patterns to help in analysis of complex surface structures. Diffraction peaks (spots), indices, reciprocal lattice vectors & different domains can all be visualised using vector graphics (e.g. PostScript).
